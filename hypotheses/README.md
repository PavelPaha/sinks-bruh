# Hypotheses workspace

Эта папка — “лабораторный стол”: **каждая гипотеза живёт в отдельной директории** с единым шаблоном файлов.

Структура каждой гипотезы:

- `hypothesis.md` — формулировка, что проверяем, какие упрощения, какой протокол
- `scripts/` — команды для сбора данных и построения графиков (воспроизводимо)
- `data/` — локальные результаты прогонов (например `.jsonl.gz`)
- `plots/` — графики именно по этой гипотезе
- `plots/manifest.json` — “индекс” графиков (описание, источник, inputs)
- `run_notes.md` — сюда вставляем результаты прогонов (цифры + наблюдения)
- `final.md` — итоговая сводка проверки гипотезы (как в paper)

Рекомендуемый порядок: H0 → H1 → H4 → H2 → H3 → H5 → H6 (см. `HYPOTHESES_TRACKER.md`).
